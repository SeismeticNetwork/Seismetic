# Seismetic Proxy

Welcome to Seismetic, an incredibly lightweight proxy built in NodeJS using tomphttp-bare and Ultraviolet. With a mere 25MB footprint, Seismetic is designed for simplicity and efficiency. This proxy is perfect for various use cases, including supporting static hosts and more.

## Features

### Lightweight
Seismetic is built with a focus on minimalism, ensuring a lightweight and efficient proxy solution. With just a 25MB size, it delivers optimal performance without unnecessary bloat.

### Support for Static Hosts
Seismetic provides robust support for static hosts, making it easy to manage and serve static content efficiently. Whether you're handling images, stylesheets, or other static files, Seismetic has you covered.

### Easy Installation
Getting started with Seismetic is a breeze. Simply use the following commands to manually install and start the proxy:

```bash
npm install
npm start
```

## Installation

To manually install Seismetic, make sure you have NodeJS and npm installed. Then, navigate to the project directory and run:

```bash
npm install
```

## Usage

Once installed, start Seismetic with the following command:

```bash
npm start
```

Seismetic will now be running, ready to handle your proxy needs.

## Configuration

While Seismetic is designed to be lightweight and straightforward, you can customize its behavior by exploring the configuration options in the source code.
